## How to use

```bash
$ clang++ -o kikyu kikyu.cpp -Iinclude -Llib.x86_64 -lcatcake -lasound -lmad -lfreetype -lpng -ljpeg -lz -lGL -lpthread -lX11 -lXxf86vm
$ ./kikyu
```
